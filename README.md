# 📊 Investment Dashboard

A modern investment dashboard built to track stocks, visualize portfolio performance, and analyze market trends.

## 🚀 Features
- 📈 Real-time stock data
- 🔍 Search with debounce optimization
- 📊 Portfolio performance chart
- ⚡ Fast and responsive UI
- 🔄 Refetch & error handling

## 🛠️ Tech Stack
- React + Vite
- JavaScript (ES6+)
- CSS Modules




## ⚙️ Installation
```bash
npm install
npm start
